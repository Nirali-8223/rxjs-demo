From Event

Creates an Observable that emits events of a specific type coming from the given event target. fromEvent(element,'click')

Interval & Timer

Interval - Creates an Observable that emits sequential numbers every specified interval of time.
Timer- After given duration, emit numbers in sequence every specified duration.Syntax :- timer(dealy,period)

of & from

Of - Emit variable amount of values in a sequence and then emits a complete notification.
From- Turn an array, promise, or iterable into an observable

ToArray

Collects all source emissions and emits them as an array when the source completes.

Custom Observable

Manually Create Observable using Obeservable.create(). There are three arguments for handling data.

Filter

Filter items emitted by the source Observable by only emitting those that satisfy a specified predicate.

Map

Apply projection with each value from source. This oprator is very usefull.

Pluck

Get the Property value from the data.it's perform short hand operation of the Map.

Tap

Used to perform side-effects for notifications from the source observable

Take

Emits only the first count values emitted by the source Observable.
